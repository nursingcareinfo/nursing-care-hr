🚀 Quick Setup Guide

1

Create Your Google Sheet

Go to sheets.google.com and click the "+" button to create a new spreadsheet. Name it "HR Daily Expenses Tracker - 2026".

2

Create 7 Sheet Tabs

At the bottom of your spreadsheet, click the "+" button 7 times to create tabs. Rename them exactly as: DASHBOARD, DAILY_EXPENSES, PETTY_CASH, EMPLOYEE_EXPENSES, BUDGET, VENDOR_MASTER, REPORTS

3

Copy Column Headers

Copy the column headers from each section below into the corresponding tabs. Start with row 1 as your header row.

4

Set Up Data Validation

Select the columns that need dropdowns, go to Data → Data Validation, and create dropdown lists for Category, Status, Payment Method, etc.

5

Add Formulas

Copy the formulas provided in each tab section. Key formulas include running balances, auto-generated IDs, and budget calculations.

6

Create Dashboard Charts

In the DASHBOARD tab, go to Insert → Chart and create pie charts for category breakdown and bar charts for weekly trends.

7

Share with Team

Click the Share button and add team members with appropriate permissions (View Only, Comment, or Edit).

📈 DASHBOARD Tab

Summary view with key metrics and charts

  • Total Budget (Monthly) - Auto-calculated
  • Total Spent (Month-to-Date)
  • Remaining Budget
  • Petty Cash Balance
  • Pending Reimbursements
  • Expenses This Week
  • Burn Rate (Daily Average)
  • Runway (Days Remaining)

💰 DAILY_EXPENSES Tab

Main transaction log for all expenses

  • Date - Transaction date
  • Expense ID - Auto: EXP-YYYY-MM-001
  • Category - Dropdown selection
  • Sub-Category - Dynamic dropdown
  • Description - Expense purpose
  • Amount (Rs.) - Expense amount
  • Payment Method - Cash/Card/Transfer
  • Status - Pending/Approved/Rejected

🪙 PETTY_CASH Tab

Imprest system for small cash expenses

  • Voucher # - PC-YYYY-MM-001
  • Particulars - What was purchased
  • Paid To - Recipient name
  • Cash Out - Disbursement amount
  • Cash In - Replenishment
  • Running Balance - Auto-calculated
  • Receipt Attached - Yes/No + Link

👥 EMPLOYEE_EXPENSES Tab

Track employee reimbursement claims

  • Request ID - REIMB-YYYY-MM-001
  • Employee Name - Claimant name
  • Employee ID - NC-XXX format
  • Department - HR/Operations/Admin
  • Amount (Rs.) - Claim amount
  • Manager Approval - Status
  • Payment Status - Pending/Processing/Paid

📊 BUDGET Tab

Monthly budget planning and tracking

  • Monthly Budget - Planned amount
  • Spent (MTD) - Month-to-date actual
  • Remaining - Budget left
  • % Used - Percentage consumed
  • Status - 🟢/🟡/🔴 indicator

🏢 VENDOR_MASTER Tab

Reference list of approved vendors

  • Vendor ID - V-001 format
  • Vendor Name - Company name
  • Category - Service category
  • Contact Person - Primary contact
  • Phone / Email - Contact details
  • Payment Terms - Cash/Net-15/Net-30
